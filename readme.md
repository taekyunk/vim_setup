# About
my vim setup using vim-plug for linux/ubuntu machines

# how to use

## Using subdirectory

1. clone this repo into ~/vimconfig
2. add symbolic shortcuts

using script

````
cd ~
sh vimconfig/setup.sh
````


manually
````
ln -s vimconfig/.vimrc .vimrc
ln -s vimconfig/.vim .vim

````
